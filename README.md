# J.A.R.V.I.S. - Just A Rather Very Intelligent System

Голосовой ассистент на Python с локальным AI (Ollama) и резервным (Gemini).

## Возможности

### Голосовое управление
- **STT**: Vosk (офлайн) или Google Speech Recognition (онлайн)
- **TTS**: Silero (CUDA) или pyttsx3 (резерв)
- **RVC**: Конвертация голоса в голос Джарвиса (опционально)

### Искусственный интеллект
- **Ollama**: Локальная нейросеть (qwen2.5)
- **Gemini**: Резервный облачный AI
- **Локальный парсер**: Быстрые команды без AI

## Установка

### Требования
- Python 3.12+
- Windows 10/11
- CUDA (для Silero TTS, опционально)
- Ollama (для локального AI, опционально)

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Настройка Ollama (опционально)
```bash
ollama pull qwen2.5:7b-instruct
```

### Настройка Vosk (опционально, для офлайн STT)
1. Скачайте модель: https://alphacephei.com/vosk/models
2. Распакуйте в `models/small/`

### Настройка Gemini (опционально)
1. Получите API ключ: https://aistudio.google.com/app/apikey
2. Добавьте в `config.yaml`:
```yaml
gemini:
  api_key: "YOUR_API_KEY"
```

## Запуск

```bash
python main.py
```

Или через батник:
```bash
start.bat
```

## Оптимизации

- **Кэширование AI**: Быстрые ответы на повторные запросы
- **Оптимизированный STT**: Улучшенное распознавание речи
- **Фоновая загрузка RVC**: Модель загружается асинхронно
- **Мониторинг производительности**: Отслеживание времени операций
- **Локальный парсер**: Мгновенное распознавание простых команд
- **LRU кэш**: Кэширование нормализации и разрешения приложений
- **Wake word кэш**: Быстрое детектирование кодового слова
- **Асинхронные действия**: Команды выполняются в фоновом потоке
- **Ускоренный AI**: Уменьшен контекст и таймауты для мгновенных ответов
- **Быстрая речь**: pyttsx3 настроен на скорость 200
- **Многопоточность Ollama**: num_thread=4 для параллельной обработки
- **GPU оптимизация**: use_mmap, num_gpu, CUDA cache
- **Предзагрузка AI**: Модель инициализируется при старте
- **CUDA оптимизация**: torch.no_grad() и empty_cache()

## Troubleshooting

### STT не работает
- Убедитесь что микрофон подключен
- Проверьте настройки конфигурации
- Для Vosk скачайте модель в `models/small/`

### TTS не работает
- Установите torch с CUDA: `pip install torch --index-url https://download.pytorch.org/whl/cu118`
- Или используйте pyttsx3 (резервный движок)

### AI не отвечает
- Запустите Ollama: `ollama serve`
- Или настройте Gemini API ключ в config.yaml

### RVC не работает
- Убедитесь что файл модели существует: `rvc_env/RVC/assets/weights/jarvis.pth`
- Проверьте что RVC окружение настроено правильно
